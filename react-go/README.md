## react-go

### 技术栈
* react
* redux
* react-router
* react-router-redux
* less
* ES6/7

### 开发:

1.安装依赖
```bash
npm install
```

2.启动本地服务
```bash
npm start
```
打开 http://localhost:3000


### 打包构建:
```bash
npm run build
```
